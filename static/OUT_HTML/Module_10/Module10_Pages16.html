Code Sample 1

What does the following code do?

The code uses the formula P=P(1+rn)nt. What does the * operator do? What does the ** operator do?

On line 8, the + operator performs addition, but what does it do on line 9? What determines which way that operator will behave?

Why do we need parentheses in line 8?

The variable name num isn't very descriptive. Can you think of a better name? Try renaming that variable (you'll need to rename it everywhere that it's used).

What do you think a variable does?

What does the = operator do?

What does line 5 do? What is the significance of the # operator?

What happens if on line 6 you replace float with int? Why is int used on line 8?

Modify the code so that it calculates the amount at the end of 10 years instead of 6.

Now try changing it so that it asks for the number of years and plugs that into the formula instead of always using the same number. Constants (values that don't change) are given names in all caps, but variables are given names in lower case.

It's possible to format the number of decimal places shown, but we won't worry about that in this course.
