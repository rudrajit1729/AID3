  • Due May 8 by 11:59pm
  • Points 20
  • Submitting a file upload
  • Available until May 11 at 11:59pm
This assignment was locked May 11 at 11:59pm.

Introduction

Backtracking is a good paradigm to know to be able to solve couple of interesting problems like N-Queen or Sudoku The greedy algorithm enables us to come up with an efficient algorithm to solve some problems. .

In question 1, you will implement a backtracking algorithm to solve a problem. (MLO1)

In question 2, you will implement a greedy technique to solve a problem. (MLO3)

What you must do

Download the assignment questions from this assignment file.

What to turn in

You will turn in the submission files via Canvas and Gradescope.

All your answers other than python code files  will be turned in through Canvas assignment in a pdf file. 

The python files to be submitted on Gradescope:

Amount.py

FeedDog.py.

Grading criteria

  1. Implement a backtracking algorithm [10 points]
    1. Describe a backtracking algorithm [2 point]
    2. Implementation [7 points]
    3. Time complexity [1 point]
  2. Implement a Greedy algorithm [10 points]
    1. Describe a greedy algorithm [2 point]
    2. Implementation [6 points]
    3. Time complexity [2 point]
