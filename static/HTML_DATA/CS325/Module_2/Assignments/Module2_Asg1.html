Introduction

We commonly see algorithms that use recursion and Divide-and-Conquer techniques. The methods to solve recurrence relations give power to us to analyze non-iterative algorithms with ease.

In question 1 and 2, you will solve recurrence relation using different methods. (MLO2)

In question 3, you will implement an algorithm using divide and conquer technique. (MLO2 and MLO3)

What you must do

Download the assignment questions from this assignment file.

What to turn in

You will turn in the submission files via Canvas and Gradescope.

All your answers other than python code files will be turned in through this Canvas assignment in a pdf file. 

The python files to be submitted on Gradescope: 

KthElement.py

Note: To access Gradescope click on 'Gradescope' in the left navigation. While submitting your code on Gradescope make sure to check file names and function names; remove any print statements from your code; and do not use any global variables, as this will result in Gradescope tests to fail. 

Grading Criteria

  1. Solve recurrence relation using three methods [ 9 points]
    1. Write recurrence relation [ 3 points]
    2. Solving using the three methods [6 points]

2. Solve recurrence relation using any one method [4 points]

    1. Problem a [ 2 points]
    2. Problem b [ 2 points]

 3. Implement an algorithm using divide and conquer technique [7 points] 

    1. Write a pseudocode/describe your strategy [1 point]

    2. Implementation [6 points]
