Exploration 1.2: Asymptotic Notations

Introduction

You will hear the word ‘asymptotic’ very often when studying any text on algorithms. The dictionary defines ‘asymptotic’ as the behavior of a line as it approaches a curve but never touches. In analyzing the efficiency of an algorithm, we are concerned with how its running time increases as the size of the input increases without bound. To compare efficiency computer scientists use three notations: O (big oh), Ω (big omega), and Θ (big theta).

In this section, we will define these asymptotic notations (MLO2) and use them with non-recursive algorithms.

In the following reading, T(n) and g(n) can be any nonnegative functions defined on positive integers.

T(n) will be the algorithm’s running time.

g(n) will be some simple function that will be used to compare T(n) with.

O Notation (Big O)

Informally, O(g(n)) is a set of all functions with a lower or same order of growth as g(n) (within a constant multiple, as n goes to infinity).

we say a function T(n) “is O(g(n))” if T(n) ∈ O(g(n)).

For example: 

  1. n ∈ O(n2) and  5n+100 ∈ O(n2) Here n is a linear function hence it has a lower order of growth than g(n) = n2.
  2. 3n(n-1) ∈ O(n2) Here 3n(n-1) is a quadratic function and it has the same order of growth as n2.
  3. But 3n(n-1) ∉ O(n) Here 3n(n-1) has a higher order of growth compared to n.

Formal Definition

A function T(n) is said to be in O(g(n)), denoted by T(n) ∈ O(g(n)), if T(n) ≤ c(g(n)) for all n ≥ n0, for some positive constant c.

How to Prove Big O bounds for a function (Optional)

Let us prove 5n + 100 ∈ O(n2). 

To prove this we have to play around the expression and select the values of c and n0. 

Here g(n)=n2, f(n)=5n+100.

To prove big O, we want to get f(n) ≤ cg(n); for this let us try to get n2 term into the equation while maintaining ≤ inequality. How about use the terms ( 5 and 100) from f(n) along with n2 as shown below.

It is true that, 5n + 100 ≤  5n2 + 100n2 ( this inequality is true for all n ≥ 1)

                ⟹ 5n + 100 ≤ 105n2 (for all n ≥ 1)

We got the equations in the form of  f(n) ≤ cg(n). So, we can select c = 105 and n0= 1

Note that this is not the only solution, you can solve it in different ways.

Question

Prove that 3n2 + 10n ∈ O(n2). Try this yourself before looking at the solution.

Click here to reveal an explanation.

Observe how we are building f(n) and g(n) relation using the terms of f(n)

Consider

                   n ≤ n2       for all n ≥ 1                   (obvious)

              10n ≤ 10n2 for all n ≥ 1                   (multiply both sides by 10)

 3n2 +10n ≤ 3n2 +10n2 for all n ≥ 1         (add 3n2 on both sides)

3n2 + 10n ≤ 13n2 for all n ≥ 1                      (Simplifying)

Hence we can choose c = 13, n0 = 1

                           3n2 +10n ≤ cn2

                           3n2 +10n ∈ O(n2)

By definition of Big-O, this proves that 3n2 + 10n ∈ O(n2).

 

 

Ω Notation (Omega)

Informally, Ω(g(n)) is a set of all functions with a higher or same order of growth as g(n) (within a constant multiple, as n goes to infinity).

we say a function T(n) “is Ω(g(n))” if T(n) ∈ Ω(g(n)).

Formal Definition

A function T(n) is said to be in Ω(g(n)), denoted by T(n) ∈ Ω(g(n)), if T(n) ≥ c(g(n)) for all n ≥ n0 , for some positive constant c.

Graphically

 

How to Prove Big Ω bounds of a function (Optional)

Let us prove 5n2 is Ω(n)

To prove this we have to select the values of c and n0. Similar to Big-O proof we will use the terms in f(n) to build the relation between f(n) and g(n).

              5n2  ≥ 5n  for all n ≥ 1 (obvious)

We can select c = 5, n0 =1

Question

Prove 2n3 - 7n +1 ∈ Ω(n3). Try this yourself before looking at the solution.

Click here to reveal the answer.

Observe how f(n) is being transformed to obtain g(n).

2n3−7n+1=n3+(n3−7n)+1 

⟹ 2n3−7n+1≥n3+1foralln≥3   

⟹ 2n3−7n+1≥n3foralln≥3 

c = 1, n0 = 3

Wondering how we get n≥3 ?

In 2n3−7n+1=n3+(n3−7n)+1 we want to make (n3−7n)≥0  to maintain our inequality in the equations that we wrote.

⟹ n2≥7 

⟹ n≥3

Θ Notation (Theta)

Informally, Θ(g(n)) is a set of all functions that have the same order of growth as g(n) ( bounded both above and below by some positive constant multiples of g(n), as n goes to infinity)

we say a function T(n) “is Θ(g(n))” if T(n) ∈ Θ(g(n)).

Formal Definition

A function T(n) is said to be in Θ(g(n)), denoted by T(n) ∈ Θ(g(n)), if c1 g(n)  ≤  T(n)  ≤  c2 g(n) for all n ≥ n0, for some positive constants c1 and c2.

Graphically

 

How to prove Big Θ bounds for a function (Optional)

Let us prove (n + 5√n)  ∈ Θ(n)

First, we will prove the right inequality (upper bound). Observe how f(n) is being transformed into g(n) satisfying ≤ inequality.

(n + 5√n)  ≤  (n + 5√n)          for all n ≥ 1

                     ≤  n + 5n                for all n ≥ 1                    (as 5√n ≤ 5n )

                    ≤  6n                         for all n ≥ 1 

Second, we prove the left inequality (lower bound). 

n ≤ (n + 5√n)                        for all n ≥ 1

Hence we have n ≤  (n + 5√n) ≤ 6n for all n ≥ 1

We can select c1 = 1, c2 =6 and n0 = 1

Prove 12n(n−1)  ∈ Θ(n2). Try this yourself before looking at the solution.

We start with f(n) equation and modify it get the inequality in terms of g(n)

⟹ 12n(n−1)≤12n2foralln≥0 

Proving Lower Bounds

12n(n−1)=12n2−12n

⟹ 12n(n−1)≥12n2−(12n)(12n) 

⟹ 12n(n−1)≥14n2foralln≥2 

Wondering, how did we get n≥2 ?

Take line 12n(n−1)≥12n2−(12n)(12n)

⟹12n2−12n≥12n2−(12n)(12n)

For this to work, following needs to be true:

12n⋅12n≥12n

Which gives, 12n≥1

⟹n≥2

Hence we have 14n2≤12n(n−1)≤12n2foralln≥2 

We can select c1 = 14 , c2 = 12 and no = 2 (note: here we picked no value to satisfy both the bounds)

Comparing Orders of Growth

Though we could use the formal definitions of O, Ω and Θ notation proofs that we discussed above to compare the order of growth of functions, a more convenient method would be to compute the limit of the ratio of the two functions in the question. For a refresher on limits refer to the Supplementary Help section at the bottom of this page.

 The limit based approach is convenient because it benefits from the power of calculus techniques. 

Note that f(n)ϵθ(g(n)) implies f(n)=O(g(n)) and f(n)=Ω(g(n)) since θ-notation is stronger notation than O-notation and Ω-notation.

The below listed two rules might be helpful to calculate limits. We will not go further into math than this. 

L'Hôpital's Rule:

limn→∞T(n)g(n)=limn→∞T′(n)g′(n)

T′(n) is derivative of  T(n)

Just refresher for you on derivatives.

If T(n)=n5 then T′(n)=5n5−1=5n4

If  T(n)=log2n then  T′(n)=1nloge2

Stirling's formula:

n!≈√2πn(ne)n for large values of n.

Example 1

Compare the order of growth of 3n2+5n+20 and n2

limn⟶∞3n2+5n+20n2=limn⟶∞(3+5n+20n2)=3

Since, the limit is equal to a positive constant, the functions have the same order of growth, 3n2+5n+20 ∈ Θ(n2)

Example 2

Show that 5n2+n ∈ Ω(n)

limn→∞5n2+nn=limn→∞5n+1=∞

This implies 5n2+n has a bigger order of growth than n, hence 5n2+n ∈ Ω(n)

Example 3

Compare the order of growth of functions log2n  and  √n

Hint: use L'Hôpital's Rule

limn→∞log2n√n=limn→∞(log2n)′(√n)′  

      =limn→∞(log2e)⋅1n12⋅√n     [since 1loge2=log2e ]

      =log2e⋅limn→∞2√n√n×√n     [since n=√n×√n ]

      =2log2e⋅limn→∞1√n  

     =2log2e×0     [Since limn→∞1√n=0 ]

     =0 

Since the limit is equal to 0, log2n has a lower order of growth than √n.

 log2n∈O(√n)

Properties Involving Asymptotic Notations

Transitivity

f(n)=θ(g(n))andg(n)=θ(h(n))implyf(n)=θ(h(n))

f(n)=O(g(n))andg(n)=O(h(n))implyf(n)=O(h(n))

f(n)=Ω(g(n))andg(n)=Ω(h(n))implyf(n)=Ω(h(n))

Reflexivity

f(n)=θ(f(n))

f(n)=O(f(n))

f(n)=Ω(f(n))

Symmetry

f(n)=θ(g(n))ifandonlyifg(n)=θ(f(n))

Transpose Symmetry

f(n)=O(g(n))ifandonlyifg(n)=Ω(f(n))
